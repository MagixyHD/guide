:::note Взаимодействия и пользователи бота

Взаимодействия - это, по-сути, вебхуки под капотом (и для ответа на взаимодействия не требуется токен бота). Таким
образом, у вас может быть приложение только для взаимодействия, доступ к которому могут получить пользователи на
сервере **без присутствия аккаунта бота**.

Поскольку `disnake` в первую очередь ориентирован на использование событй шлюза, вам всё равно потребуется аккаунта
бота. Вы можете проверить другие библиотеки, такие как [`hikari`](https://www.hikari-py.dev/) для использования этой
функции.

:::
